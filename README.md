# Git Öğreniyorum Projesi

Bu benim GitHub öğrenirken oluşturduğum **ilk projemdir**.
Şu an Git komutlarını ve GitHub bağlantısını test ediyorum.

## Neler Öğrendim?
* Git kurulumu ve konfigürasyonu
* Repository (Depo) mantığı
* `add`, `commit` ve `push` komutları
* Versiyon takibi

## Gelecek Hedeflerim
1. Branch (dal) mantığını öğrenmek.
2. Merge ve Rebase farkını anlamak.
3. Teknofest projem için bu yapıyı kullanmak.
